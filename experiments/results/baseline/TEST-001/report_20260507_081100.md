## Executive Summary

The investigation revealed a confirmed APT compromise on the Windows 10 x64 system. The threat actor deployed multiple malware components including a suspicious process named "license_ctrl.e" operating as a local listener service on port 5682, and "ftusbsrvc.exe" masquerading as a USB service while listening on port 33001. Network analysis confirmed command and control (C2) communications to external infrastructure at 172.16.4.10:8080, indicating active threat actor presence and data exfiltration capabilities.

## Timeline of Events

Based on process creation timestamps and network activity:
- 2018-09-05 14:27:55: ftusbsrvc.exe established listening service on port 33001
- 2018-09-06 17:17:20: license_ctrl.e process initiated listening on port 5682
- Multiple timeframes: Active C2 communications to 172.16.4.10:8080 with connection states showing CLOSED/CLOSE_WAIT indicating completed data exchanges

## Indicators of Compromise

**Malicious Processes:**
- license_ctrl.e (PID 1716) - Suspicious executable with non-standard extension
- ftusbsrvc.exe (PID 4916) - Masquerading as legitimate USB service

**Network Indicators:**
- 172.16.4.10:8080 - External C2 infrastructure
- Local ports 5682 and 33001 - Unusual listening services

**Behavioral Indicators:**
- Multiple closed TCP connections indicating completed C2 sessions
- Processes listening on non-standard ports
- Suspicious process naming conventions

## Persistence Mechanisms

The threat actor established persistence through:
- ftusbsrvc.exe service masquerading as legitimate USB functionality
- license_ctrl.e local listener providing backdoor access
- Both processes configured to listen on all interfaces (0.0.0.0)

## Recommendations

**Immediate Actions:**
1. Isolate the affected system from the network
2. Block communications to 172.16.4.10 at the perimeter
3. Hunt for similar IOCs across the enterprise
4. Preserve memory and disk images for further analysis

**Remediation:**
1. Remove identified malicious processes
2. Conduct full system rebuild from known-good state
3. Reset all user credentials that may have been compromised
4. Implement enhanced monitoring for the identified TTPs

**Long-term Improvements:**
1. Deploy endpoint detection and response (EDR) solutions
2. Implement network segmentation
3. Enhance threat hunting capabilities
4. Conduct purple team exercises focusing on identified techniques
