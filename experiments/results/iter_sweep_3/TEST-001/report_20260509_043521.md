## Executive Summary

This Windows 10 system was compromised by an APT actor who deployed multiple malicious processes establishing command and control communications and backdoor access. The attack involved deployment of suspicious executables `license_ctrl.e` and `ftusbsrvc.exe` that created listening services on ports 5682 and 33001, while communicating with a command and control server at 172.16.4.10:8080.

## Timeline of Events

- **2018-09-05 14:27:55**: `ftusbsrvc.exe` process created, established listening service on port 33001
- **2018-09-06 17:17:20**: `license_ctrl.e` process created, established listening service on port 5682
- **Multiple timestamps**: Active C2 communications observed to 172.16.4.10:8080

## Indicators of Compromise

- **Malicious Processes**: `license_ctrl.e` (PID 1716), `ftusbsrvc.exe` (PID 4916)
- **C2 Communications**: 172.16.4.10:8080 (multiple TCP connections)
- **Backdoor Ports**: TCP/5682 (license_ctrl.e), TCP/33001 (ftusbsrvc.exe)
- **Suspicious Process**: `usbclient.exe` (PID 6648) running from explorer.exe

## Persistence Mechanisms

The malware established persistence through deployment of backdoor services listening on high-numbered ports, allowing remote access and command execution capabilities.

## Recommendations

1. Immediately isolate the compromised system
2. Block C2 IP 172.16.4.10 at network perimeter
3. Hunt for similar processes across the network environment
4. Implement monitoring for processes listening on unusual high-numbered ports
5. Conduct full forensic imaging before remediation
